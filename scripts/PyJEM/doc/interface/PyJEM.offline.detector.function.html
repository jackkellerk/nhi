PyJEM.offline.detector.function module¶

class PyJEM.offline.detector.function.Detector(detectorName)¶

Bases: object

It can acquire the information, acquire image data, execute automatic function, etc. of detector specified by argument.
Detector names to be entered as arguments can be obtained with “detector.detectorlist” or “detector.get_attached_detector()”.
